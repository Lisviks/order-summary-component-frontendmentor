### What I learned

To create CSS variables, first declare them on the `:root` pseudo-class, which will allow to use them globally.

```css
:root {
  --pale-blue: hsl(225, 100%, 94%);
}
```

Use CSS variables

```css
.container {
  background-color: var(--pale-blue);
}
```
